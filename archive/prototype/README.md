# Prototyp-Archiv

Dieses Verzeichnis enthält frühere, explorative Projektstände.

## Inhalt

- `DesktopSafe_Prototype_Initial.zip`  
  Früher, stark kommentierter Prototyp einer C#-Desktop-Anwendung. Dieser Stand ist **nicht** automatisch deckungsgleich mit dem späteren Pflichtenheft, der finalen Architektur oder dem geplanten Release-Schnitt.

## Zweck

- Referenz für frühe Ideen
- Vergleich alter und neuer Architekturentscheidungen
- mögliche Wiederverwendung einzelner Gedanken oder UI-Ansätze
