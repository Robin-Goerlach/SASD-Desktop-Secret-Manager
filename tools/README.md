# Tools

Dieses Verzeichnis ist für Hilfswerkzeuge und projektspezifische Hilfsmittel vorgesehen.

Mögliche Inhalte in späteren Ausbaustufen:

- Build- und Release-Helfer
- Konvertierungswerkzeuge
- Testdaten-Generatoren
- Wartungs- und Prüfskripte
- Hilfsprogramme für Import, Export oder Migrationsaufgaben

Im aktuellen Entwicklungsstand spielt dieses Verzeichnis noch keine zentrale Rolle, ist aber als geplanter Sammelpunkt für unterstützende Werkzeuge bereits vorgesehen.
