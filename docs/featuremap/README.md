# Featuremap

Die Featuremap ordnet die Produktfunktionen in V1, V1.x, V2 und spätere Stufen ein.

## Führende Datei

- `Featuremap_SASD_Desktop_Secret_Manager.md`

## Zweck

Die Featuremap verhindert, dass gute Ideen ungeordnet in den V1-Kern gedrückt werden. Sie macht sichtbar, welche Funktionen Kernumfang sind, welche Härtung darstellen und welche später bewusst nachgezogen werden.
