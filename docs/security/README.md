## Grundsatz

Das Produkt ist sicherheitskritisch. Es darf keine falschen Sicherheitsversprechen machen. Es schützt lokale verschlüsselte Tresordateien, sichere Bedienabläufe und kontrollierte Geheimnisbehandlung, kann aber kein kompromittiertes Betriebssystem vollständig neutralisieren.
