# Roadmap

## V1

Ziel: stabiles Fundament.

- internes Tresorformat `.svault`
- lokaler Einzelbenutzerbetrieb
- Master-Passwort
- mehrere unabhängige Tresore
- Gruppen
- Tags
- Suche und Filter
- Eintragstypen
- flexible Zusatzfelder
- Passwortgenerator
- Auto-Lock
- Clipboard Auto-Clear
- atomisches Speichern
- verschlüsselte Backups
- `.psafe3`-Import
- Tests für Security, Storage und Mapping-Grundlagen

## V1.x

Ziel: Härtung, Komfort, Export, Strukturgewinn.

- `.psafe3`-Export
- Kompatibilitätsbericht
- Vorlagen pro Typ/Gruppe
- Generatorprofile pro Gruppe
- besseres Mehrtresor-Handling
- zusätzliche Tests
- UI-Feinschliff
- Migrations- und Recovery-Vorbereitung

## V2.0

Ziel: ausgebauter professioneller Betrieb.

- Passwort-Historie
- spätere Recovery-Strategie ohne Backdoor
- erweiterter Password-Safe-Kompatibilitätsmodus
- zusätzliche Verwaltungs- und Komfortfunktionen
- fortgeschrittene Mehrtresor-Funktionen
