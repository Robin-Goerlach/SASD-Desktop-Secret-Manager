# Archiv

Dieses Verzeichnis ist für ältere Dokumentfassungen vorgesehen, die nicht mehr führend sind, aber nachvollziehbar erhalten bleiben sollen.

Aktuell enthält dieses ZIP keine alten Originaldokumente aus dem Repository. Es wird empfohlen, den bisherigen `docs`-Ordner vor dem Ersetzen lokal zu sichern, falls ältere PDF-/Word-Fassungen später noch nachvollzogen werden sollen.
