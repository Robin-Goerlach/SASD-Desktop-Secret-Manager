# Import, Export und Interoperabilität

Dieses Verzeichnis sammelt Konzepte zu Datenübernahme, Export, Password-Safe-Interop und späteren Migrationsfunktionen.

## Dokumente

- `password-safe-interop-concept.md`

## Grundsatz

Import und Export sind sicherheitskritische Funktionen. Besonders unverschlüsselte Exportformate wie CSV dürfen nie beiläufig angeboten werden, sondern benötigen Warnungen, bewusste Nutzerentscheidung und klare Lösch-/Bereinigungsregeln.
