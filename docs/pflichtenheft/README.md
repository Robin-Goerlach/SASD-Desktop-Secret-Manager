# Pflichtenheft

Dieses Verzeichnis enthält das aktuelle Pflichtenheft als Umsetzungsspezifikation für den SASD Desktop Secret Manager.

## Führende Datei

- `Pflichtenheft_Desktop_Secret_Manager_SASD.md`

Das Pflichtenheft konkretisiert V1, V1.x, V2 und spätere Stufen. Es beschreibt Datenmodell, UI, Sicherheitsregeln, `.svault`, Interop, Testpflichten und die Umsetzung der HTTPS-/TLS-Zertifikatsverwaltung.
