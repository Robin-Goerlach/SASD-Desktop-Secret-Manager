# Kommentierter Code-Stand

Dieser ZIP-Stand basiert auf dem aktuellen öffentlichen Repository-Stand der angegebenen GitHub-URL.

Ziel dieser Fassung ist nicht eine funktionale Neuentwicklung, sondern eine **lesbarere, stärker dokumentierte Arbeitsfassung**:

- zusätzliche XML-Dokumentation bei zuvor unkommentierten öffentlichen Klassen und Methoden
- erläuternde `//`-Kommentare an ausgewählten Kernstellen
- sprechende Dateiüberblicke in den C#-Dateien
- testseitige Kommentare, damit die Testnamen noch klarer als ausführbare Spezifikation gelesen werden können

Die vorhandene fachliche Logik wurde bewusst nicht umgebaut.
